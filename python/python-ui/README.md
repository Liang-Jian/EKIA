# web UI自动化框架


### 设计意图
使用JS为主、css为辅助。包含PO思想、数据分离、设计模式的测试框架。

公司因为历史原因使用了elementui + vue 开发了CMS系统。但是vue对jquery的支持有限，
且原生selenium 无法支持多样的现代前端框架。所以本人使用了js+ python进行了个性化适配。

使用单例模式，减少无用实例化对象。可灵活使用混合原生，封装方法，
